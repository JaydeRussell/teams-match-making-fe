# What is this?
This is a project designed to streamline pairings in warhammer 40k teams events. This specific repo is for the front-end project. The backend portion can be found [here](https://github.com/JaydeRussell/teams-match-making-be)


# Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


# linting

```bash
npm run lint
```
